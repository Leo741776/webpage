DES File Encrypter

A desktop application with a GUI, used for securing files via the Data Encryption Standard (DES). I used a from-scratch implementation of an existing DES algorithm, including a full Feistel network, 16-round key scheduling, and S-box substitutions. To ensure strong security for real-world files, this program makes use of block cipher mode of operation and PKCS#7 padding, allowing the user to safely encrypt and decrypt files by using a 64-bit hex key generated through the UI.

2D Arcade Shoot 'Em Up

A vertical arcade shoot 'em up that features a dynamic game loop via the JavaFX AnimationTimer to manage combat and scaling difficulty at its core. Lightweight Data Structures Library for C

A library that provides a collection of basic data structures - Stack, Queue, Linked List, Hash Table, and Max Heap - focusing on memory efficiency. It implements dynamic memory management across the board, making use of realloc for array-based structures and strdup() for key ownership in the hash table to ensure data stays intact. I've included O(1) optimizations like tail pointers for linked list insertions and circular buffer logic (via modulo) for queues to prevent the O(n) cost of shifting elements. The hash table makes use of separate chaining and the djb2 algorithm, triggering a full rehash to a load factor of 0.7 to maintain performance. I tried to stick to a clean API pattern that was possible with C, so I included creation and destruction via _create() and _free() functions. I also tried to keep consistent error-handling by using NAN returns for invalid operations and integer status codes for memory failures.

  • Language: C
